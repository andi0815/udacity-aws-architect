# Terraform

## Install terraform
```shell
brew tap hashicorp/tap
brew install hashicorp/tap/terraform
```
### ...for openstack plugin (not needed here...)
```shell
brew install go@1.17
```

# Setup
AWS Account settings must be defined in ~/.aws/credentials
```toml
[default]
aws_access_key_id=
aws_secret_access_key=
aws_session_token=
```
